Html

ماهي Html ؟

HTML هي اختصار لـ Hyper Text Markup Language وتعني لغة ترميز النص التشعبي. وهي اللغة المسؤولة عن بناء هيكل وبنية مواقع الويب

لماذا سميت بأسم الترميز

لأنها تستخدم وسوم (Tags) لترميز وصف المحتوى. مثلًا: هذا عنوان، هذه فقرة، هذا رابط

تاريخها

إصدار HTML في سنة 1993 من قبل العالم Tim Berners-Lee، مؤسس الويب العالمي.

ملحوظه : هي ليست لغة برمجة ! هي لغة ترميز وصفية ، أي أنها تصف محتوى الصفحة فقط . لا تحتوي على منطق أو حسابات

مكونات صفحة HTML

عناصرها الأساسية

  • html : الوسم الجذر. يحتوي كل الصفحة. يجب أن يكون أول وسم وآخر وسم
  • head : رأس الصفحة. يحتوي الإعدادات والمعلومات غير المرئية (metadata, title, CSS links).
  • body : جسم الصفحة. يحتوي كل المحتوى المرئي (نصوص، صور، روابط، إلخ)

أدوات مهمة لتعليم اللغة :

1- برنامج محرر النصوص

هو برنامج لكتابة الكود و يوفر مميزات مثل :

  • تلون الكود
  • إكمال تلقائي
  • أظهار الأخطاء

الأختبارات

  1. VS Code : الأشهر والأفضل (مجاني)
  2. Atom : من GitHub (مجاني)
  3. Sublime Text : سريع وخفيف

2- المتصفح (Browser)

المتصفح هو مترجم HTML. يقرأ ملف HTML ويحوله لصفحة ويب مرئية. كما يحتوي على DevTools (اضغط F12) لفحص وتحرير الكود مباشرة

أهم المتصفحات

  • Google Chrome : الأفضل للمطورين
  • Firefox : ممتاز ويحترم الخصوصية
  • Edge : من Microsoft، بنفس محرك Chrome

عناصر الـ Head

عناصر داخل head لا تظهر في الصفحة مباشرة، لكنها مهمة جداً للمتصفح ومحركات البحث

الـ Meta Tags

  1. charset : ترميز الحروف (UTF-8 يدعم العربية)
  2. viewport : للتصميم المتجاوب على الموبايل
  3. description : وصف الصفحة (يظهر في Google)
  4. keywords : الكلمات المفتاحية (أقل أهمية الآن)
  5. author : اسم المؤلف

Java Script

ما هي JavaScript؟

JavaScript هي لغة برمجة عالية المستوى، ديناميكية، ومفسرة تستخدم بشكل أساسي لجعل صفحات الويب تفاعلية. تعمل JavaScript في المتصفح وعلى الخادم (Server) باستخدام Node.js. هي واحدة من اللغات الثلاث الأساسية لتطوير الويب:

لماذا سميت بأسم"الترميز

  • html : هيكل الصفحة (Structure)
  • CSS: تنسيق وشكل الصفحة (Style)
  • body : جسم الصفحة. يحتوي كل المحتوى المرئي (نصوص، صور، روابط، إلخ)

لماذا JavaScript مهمة جداً؟

1- اللغة الوحيدة التي تعمل في المتصفح كل المتصفحات الحديثة (Chrome, Firefox, Safari, Edge) تفهم JavaScript فقط. لا يمكنك استخدام Python أو Java أو C++ مباشرة في المتصفح، لكن JavaScript تعمل مباشرة بدون أي إضافات!

2- تستخدم في كل موقع تقريباً انظر حولك في الإنترنت:

  • Facebook: التفاعلات (Like, Comment, Share) كلها JavaScript
  • YouTube: مشغل الفيديو، التعليقات، الاقتراحات
  • Gmail: إرسال البريد بدون تحديث الصفحة
  • Google Maps: التحريك والتكبير التفاعلي

ما الفرق بين JavaScript vs Java

Java

  • لغة سكريبت (Script) للويب
  • تعمل في المتصفح مباشرة
  • ديناميكية (لا تحتاج Compilation)
  • سهلة التعلم للمبتدئين
  • الأنواع ديناميكية (Dynamic Typing)

Java Script

  • لغة برمجة عامة (General Purpose)
  • تحتاج JVM للتشغيل
  • تتطلب Compilation قبل التنفيذ
  • أكثر تعقيداً للمبتدئين
  • الأنواع ثابتة (Static Typing)

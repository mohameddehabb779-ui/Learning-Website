CSS

ما هي CSS3؟

CSS هي اختصار لـ Cascading Style Sheets ومعناها أوراق الأنماط المتتالية. بينما HTML تبني الهيكل العظمي للموقع، CSS تضيف الألوان والتنسيق والتصميم الجمالي

أدوات التعلم

  • محرر النصوص : VS Code, Atom, Sublime Text
  • متصفح حديث : Chrome, Firefox, Edge.
  • DevTools :اضغط F12 للتجربة المباشرة

طرق ربط CSS بـ HTML

1. Inline CSS (داخل العنصر مباشرة)

طريقة كتابة CSS مباشرة داخل وسم HTML باستخدام السمة style. هذه الطريقة سهلة وسريعة للتجربة لكنها غير مناسبة للمشاريع الكبيرة.

متى نستخدمه؟

  • للتجربة السريعة فقط
  • عندما تحتاج لتنسيق عنصر واحد بشكل استثنائي
  • في البريد الإلكتروني (Email templates)

العيوب:

  • يصعب تعديل التنسيقات لأنها متناثرة في كل مكان
  • لا يمكن إعادة استخدام الكود
  • يصعب الصيانة في المشاريع الكبيرة
صوره

2. Internal CSS (داخل ملف HTML)

كتابة CSS داخل وسم style في قسم head من ملف HTML. مناسب للصفحات الصغيرة أو عندما يكون لكل صحة تنسيقات خاصة بها

متى نستخدمه؟

  • صفحات HTML منفردة (Single Page)
  • عندما لا تريد إنشاء ملف CSS منفصل
  • للنماذج الأولية (Prototypes)

3.External CSS (ملف منفصل - الأفضل)

إنشاء ملف منفصل بامتداد .css وربطه بملف HTML باستخدام وسم link. هذه هي الطريقة المثلى والأكثر استخداماً

المميزات

  • فصل كامل بين المحتوى (HTML) والتصميم (CSS)
  • عإمكانية استخدام نفس ملف CSS لعدة صفحات HTML
  • سهولة الصيانة والتعديل من مكان واحد
  • تخزين مؤقت (Caching) يحسن أداء الموقع
  • تنظيم أفضل للمشروع

كيفية الربط:

  1. rel="stylesheet":يحدد أي عنصر HTML سيتم تنسيقه (مثل p, h1, .class, #id)
  2. Declaration Block (كتلة التصريح):محاطة بأقواس معقوفة { } تحتوي على الخصائص
  3. Property (الخاصية):ما تريد تغييره (color, font-size, background)
  4. Value (القيمة):القيمة الجديدة (red, 20px, #ffffff)
  5. Declaration (التصريح):Property + Value ينتهي بفاصلة منقوطة ;

البنية الأساسية لـ CSS

  • Selector (المحدد): يحدد أن هذا الملف هو ورقة أنماط
  • href:مسار ملف CSS (يمكن أن يكون نسبي أو مطلق)
